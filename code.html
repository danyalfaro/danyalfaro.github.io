Light's Always Out

Computer Vision Project

Overview

Using Python and OpenCV, I challenged myself to create a solver for any game of “Light’s Out” given an image of the board. Implemented a step-by-step solutions method to provide the user with easy to follow instructions.

Motive

Even though it started as a course assignment to write a python script that solves a "Light's Out" game board, the user always had to input the board as an array. Finding this process tedious, I took a crash course on computer vision with OpenCV and challenged myself to integrate the library with the python script to solve the board automatically.

Development

Python was the language chosen for this task, since using Numpy and OpenCV libraries provided many of the tools for this project. The program runs in the command line in which an argument parser was added to input the image of the board.

uav Search and Rescue

Software Division Leader

Overview

Currently the Software Division Leader in the research and development of an eVTOL Aircraft for use in Search and Rescue.

Motive

These past years, Puerto Rico has lived through many catastrophes including hurricanes and earthquakes. With the amount of uncertainty on what could hit next, we are determined to research and apply new technologies in the creation of an Electrical Vertical Takeoff Landing Aircraft capable of recompiling data in hopes of finding people in need of help.

Development

Python has been the primary language of the project. By utilizing Flask, OpenCV, TensorFlow and Keras inside of an NVidia TX2, we are able to receive realtime video input, perform image segmentation and classification, and broadcast via network into a server with authentication to avoid unwanted attention.
